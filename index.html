PyTouch

A Machine Learning Library for Touch Processing

Get Started

Rapid Experimentation

Use out of the box pre-trained models relating to touch sensing directly in your experiments. Focus on what matters through re-usable building blocks provided by PyTouch.

Extendable

Use PyTouch as a base for all future experiments through extending the library with additional tasks and models.

Powered by PyTorch

PyTouch is built upon PyTorch which allows for rapid experimentation with projects related to touch sensing and manipulation. PyTouch is easily extendable via the PyTorch library.
